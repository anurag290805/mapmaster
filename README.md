🌍 MapMaster

MapMaster is an interactive browser-based geography learning game that helps users test and improve their knowledge of countries, states, capitals, and flags through multiple engaging game modes.

Play directly in your browser and challenge yourself across world and country maps with real-time scoring and progress tracking.

🔗 Live Demo:
https://anurag290805.github.io

✨ Features
🌎 Multiple Map Regions

World map (190+ countries)

Country maps (India, USA)

Continent-based quizzes

🎮 Multiple Game Modes

Typing Mode – Type the name of the region from memory

Locate on Map – Click the correct location on the map

Flags Mode – Identify countries from their flags

Capitals Mode – Name the capital city of each region

📊 Gameplay Systems

Real-time score tracking

Streak system for consecutive correct answers

Progress bar showing completion

Hint system

Timer and lives in challenge mode

🎨 UI & Experience

Clean modern UI with animated transitions

Light / Dark theme toggle

Sound effects and confetti win animation

Interactive SVG-based maps

Fully responsive layout

The game uses dynamic DOM manipulation and SVG interaction to highlight regions, track progress, and manage gameplay state.

⚙️ How It Works

MapMaster loads different SVG maps dynamically depending on the selected region. Each map contains labeled paths representing countries or states.

The game engine then:

Loads the selected region dataset

Tracks user guesses

Highlights correct regions

Updates score, progress, and streak

Determines win/lose conditions
