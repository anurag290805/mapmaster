✨ Overview

MapMaster is an interactive browser-based geography quiz game designed to help users improve their knowledge of global geography.

The platform allows players to identify countries, states, capitals, and flags through different gameplay modes while tracking progress, streaks, and accuracy.

The project focuses on interactive SVG maps, smooth UI animations, and an engaging learning experience.

🎮 Game Modes
✍️ Typing Mode

Players type the name of the correct region from memory.

🎯 Locate on Map

Click the correct location directly on the interactive map.

🚩 Flag Mode

Identify countries based on their flags.

🏛 Capitals Mode

Name the capital city corresponding to each region.

⚙️ How It Works

The user selects a region and game mode

The game loads the appropriate SVG map

Regions are tracked using ARIA labels

Correct answers highlight regions on the map

Score, streak, and progress update dynamically

The game ends when all regions are completed or time/lives run out
