# 🌍 MapMaster

**MapMaster** is an interactive geography learning game where players identify regions on maps using multiple gameplay modes.

Built with **HTML, CSS, and JavaScript**, the game uses **SVG maps** to create a smooth and interactive geography quiz experience.

---

## 🎮 Features

* 🗺 **Multiple Maps**

  * India
  * USA
  * Continents
  * World

* 🎯 **Game Modes**

  * Typing Mode
  * Locate Mode
  * Flag Mode
  * Capital Mode

* ⚡ **Challenge System**

  * Timed challenges
  * Limited lives
  * Score tracking

* 🔥 **Gameplay Enhancements**

  * Streak tracking
  * Hint system
  * Review mode for missed regions
  * Zoomable and pannable maps
  * Sound effects
  * Dark mode

* 🥚 **Hidden Easter Egg**

  * Bonus points for identifying India's Union Territories

---

## 🧠 How to Play

1. Choose a map (Country, Continent, or World).
2. Select a game mode.
3. Play in **Practice** or **Challenge** mode.
4. Identify regions correctly to increase your score and streak.
